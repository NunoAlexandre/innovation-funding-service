Collaboration agreement

Before your project can start, all consortium members must sign a collaboration agreement.

This agreement is between project participants and should cover how the project will operate. It should also include how you will exploit the results of the project. Please allow enough time to complete this process before your project start date.

The following partners need to sign this collaboration agreement:

  • Empire Ltd
  • Ludlow
  • EGG

An example of collaboration agreement can be found on the Lambert Agreement website.
