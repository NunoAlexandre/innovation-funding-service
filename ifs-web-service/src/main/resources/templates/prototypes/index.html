Sprint 1. Application journey completion

Application refinement

  • As an applicant I want to view all partners project finances to be able to assess and compare within the overall project costs
  • As an applicant I want to view all organisations and individuals working on my application
  • As a lead applicant I want to invite other collaborators during the application phase so that my full team can work on the application (even if undecided when application created)
  • As an applicant I want to print/download my application so that I can review and work offline
  • As a collaborator I want to complete my section of the form and mark this as done so that the lead applicant is able to track progress and submit when ready
  • Plus style refinements - e.g. consistency of page headings, layout of forms, restructure of overview page, changing accordions etc.

Academic finances

  • As an academic contributer I need to document my financial requirements to provide to the relevant organisations
  • As an academic contributer I need to provide these finances (J-ES) to Innovate UK to submit my grant claim
  • As a partner I need visibilty of the academic finance contribution to my application so that I'm able to understand the total project costs
  • As an assessor I want visibility of all academic partner contributions so that I can review the breakdown of the full project costs
  • As a member of the project finance team at Innovate UK I need access to access academic finances as part of the due diligence phase to review these as part of the overall project costs
